# Precio Bitcoin en Tiempo Real

Muestra el precio del Bitcoin en tiempo real con actualizaciones automáticas.

## Características

- Precio actual del Bitcoin (BTC/USD)
- Actualizaciones automáticas cada minuto
- Cambio porcentual en 24 horas
- Diseño responsive y minimalista

## Demo

https://mike-0-code.github.io/precio-bitcoin/

## Tecnologías

HTML, CSS y JavaScript. Datos de CoinGecko API.

## Instalación

Abre index.html en tu navegador.
